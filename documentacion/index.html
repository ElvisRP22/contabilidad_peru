Arquitectura del Sistema

El Sistema Contable Perú es una aplicación web multi-tenant desarrollada en Django (backend) y React (frontend), diseñada para estudios contables en Perú.

Backend

  • Framework: Django 5.x + DRF
  • Base de datos: PostgreSQL 15
  • Auth: JWT (SimpleJWT)
  • API: REST

Frontend

  • Framework: React 18 + TypeScript
  • Build: Vite
  • State: Zustand
  • UI: TailwindCSS

Módulos

  • Core: Auth, Empresas
  • Contabilidad: PCGE, Asientos
  • Facturación: CPE, SUNAT
  • Inventario: Productos, Kárdex
  • Nómina: Empleados, Planillas
